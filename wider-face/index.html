wider-face 数据集

  • 官网地址

  • 第三方说明地址

  • 总共有 32203 张图片（训练集12880、验证集3226、测试集16097），393,703 个人脸

  • 总共有61个类别，每个类别里面随机选取 40%、10%、50% 作为训练集、验证集、测试集

  • wider_face_split 为原始标签、scrfd_label 为 scrfd修改后的标签（对格式进行了修改，训练集加上了landmarks）

  • 训练集中没有人脸目标的样本：

  • 标签说明

    File name
Number of bounding box
x1, y1, w, h, blur, expression, illumination, invalid, occlusion, pose

    实际抠出来看发现只有 blur 和 occlusion 比较准确，通过这两项可以过滤掉2/3的无效人脸

    # !usr/bin/env python
# -*- coding:utf-8 -*-

import os
import cv2 as cv
from tqdm import tqdm


def split_content(content):
    totoal_line = len(content)
    sub_content_map = {}
    idx = 0
    while idx < totoal_line:
        im_name = content[idx]
        bbox_num = int(content[idx + 1])
        if bbox_num == 0:
            bbox_num = 1

        bbox_info = content[idx + 2:idx + 2 + bbox_num]
        idx = idx + 2 + bbox_num
        sub_content_map[im_name] = bbox_info
    print(idx)
    return sub_content_map


def read_label_info(labelfile):
    with open(labelfile) as f:
        content = f.readlines()
        content = [line.strip() for line in content]
        return split_content(content)

def save_label(save_name, img_face_info):
    with open(save_name, 'w') as f:
        for k, v in img_face_info.items():
            if len(v) == 0:
                continue

            f.write(k)
            f.write('\n')

            for line in v:
                f.write(line)
                f.write('\n')
